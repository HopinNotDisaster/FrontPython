Django 1.5.1 版本发行说明¶

2013 年 3 月 28 日

这是 Django 1.5.1，是 Django 1.5 的修复错误版本。它与 Django 1.5 完全向后兼容，但包含了一些修复。

最重要的修复是针对 Django 1.5 引入的内存泄漏。在某些情况下，反复迭代查询集可能会导致内存泄漏 - 有时会泄漏相当多的内存。如果您想了解更多信息，详细信息可以在我们的 问题跟踪器 <19895>`（以及 Python 自身的 :bpo:`相关问题）中找到。

如果您在 Django 1.5 下遇到了内存问题，升级到 1.5.1 版本应该可以解决这些问题。

Django 1.5.1 还包含了一些较小的修复：

  • 在测试期间发出的模块级警告不再被默默隐藏 (#18985)。
  • 防止在用户管理界面中对密码哈希进行过滤 (#20078)。
